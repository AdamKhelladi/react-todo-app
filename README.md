# React To-Do App

A simple and interactive To-Do List built with **React** and **localStorage**.

## ✨ Features

- ✅ Add tasks
- ✏️ Edit tasks
- ❌ Delete tasks
- 💾 Save tasks using localStorage (persists after refresh)

## 🚀 Getting Started

```bash
git clone https://github.com/AdamKhelladi/react-todo-app.git
cd react-todo-app
npm install
npm start
